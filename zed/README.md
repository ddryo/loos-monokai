# LOOS Monokai for Zed

[Zed](https://zed.dev)向けにカスタマイズしたMonokaiテーマです。

## Themes

- **LOOS Monokai** (dark)

## Dev Extension

1. このリポジトリをクローン
2. Zedのコマンドパレットで`zed: install dev extension`を実行
3. この`zed/`ディレクトリを選択

その後`settings.json`でテーマを指定:

```json
{
  "theme": {
    "mode": "dark",
    "dark": "LOOS Monokai"
  }
}
```
